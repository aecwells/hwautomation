The HTML documentation has not been built yet. You can build it using Sphinx or view the Markdown documentation directly.

Build HTML Documentation

To build the HTML documentation:

# Install documentation dependencies
                        
pip install -r requirements.txt


# Navigate to docs directory
cd docs/

# Build HTML documentation
make html

# Serve locally (optional)
make serve

After building, refresh this page to access the HTML documentation.

Docker/Container Setup

If running in a container, you can build documentation as part of the container:

# Add to Dockerfile
RUN cd docs && make html

# Or build in running container
docker compose exec app bash -c "cd docs && make html"
